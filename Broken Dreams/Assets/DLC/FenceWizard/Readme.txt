To open the window goto Tools/FenceAndPathway/FenceWizard

1 :Assign which prefab you like to place

2 :click "Begin Editing"

3 :Click where you would like the first end to be and drag to align the second end.

4 :when your finished click "End Editing"

-Details

Continuous mode will allow you to drag and place multiple prefabs.


Settings:

-Additional Vertical Offset : how much vertical offset to apply to the first end.
	
-Extra Height : how much extra height is applied to the second end.

-Extra Length : how much extra length to allow while dragging before placing a new object in continuous mode. this is useful if you want more space between pathways for example.
